# Snake Game SDL3

A classic Snake game implementation using C++ and SDL3.

## Build

```bash
cmake -S . -B build
cmake --build build
```

## Run

```bash
./build/SnakeSDL3
```

## Clean

```bash
rm -rf build
```

## Project Structure

- `include/` - Header files
- `src/` - Implementation files
- `CMakeLists.txt` - Build configuration

## Requirements

- C++11 or later
- CMake 4.0+
- SDL3
